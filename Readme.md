
# Youtube Backend

## Run Locally

Clone the project

```bash
  git clone https://github.com/pratyushsingha/youtubeBackend
```

```bash
  cd youtubeBackend
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
server is  running at `http://localhost:8080/`
